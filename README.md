### Скопируйте репозиторий:
C помощью команды в терминале
`git clone https://github.com/skypro-008/lesson23-and-tests.git`

Откройте склонированный репозиторий в PyCharm.

### Создайте виртуальное окружение:
С помощью команды в терминале
mac os
`python3 -m venv env`,
windows
`python -m venv env`
`source env/bin/activate`, где env - название окружения

#### Установка зависимостей:
С помощью команд в терминале
`pip install -r requirements.txt`
